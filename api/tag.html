laonlp.tag

Modules

laonlp.tag.pos_tag(words: List[str], engine: str = 'perceptron', corpus: str = 'SeqLabeling') → List[Tuple[str, str]]

Lao Part-of-speech

We use lao corpus from https://github.com/FoVNull/SeqLabeling

Parameters:

  • words (List[str]) – a list of tokenized lao words

  • engine (str) – engine of pos_tag (perceptron engine)

  • corpus (str) – SeqLabeling (corpus from https://github.com/FoVNull/SeqLabeling) or yunshan_cup_2020 (corpus from https://github.com/GKLMIP/Yunshan-Cup-2020)

Returns:

a list of tuples (word, POS tag)

Return type:

List[tuple[str, str]]

Example:

from laonlp.tokenize import word_tokenize
from laonlp.tag import pos_tag

sent = word_tokenize("ພາສາລາວໃນປັດຈຸບັນ.")
pos_tag(sent)
# output: [('ພາສາລາວ', 'N'), ('ໃນ', 'PRE'), ('ປັດຈຸບັນ', 'ADJ'), ('.', 'PUNCT')]
