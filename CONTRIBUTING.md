# Contributing to LaoNLP

Hi! Thanks for your interest in contributing to LaoNLP.

LaoNLP is a Lao language Natural Language Processing toolkit. I want to create open source toolkit for Lao language like [PyThaiNLP](https://github.com/PyThaiNLP/pythainlp) in Thai language.

This project use git as version control system and use github as git hosting. You can fork and sent me a pull request to the project.

## Ideas

If you has any ideas to improve this tools or the function that you want, you can create a issues at [https://github.com/wannaphong/LaoNLP/issues](https://github.com/wannaphong/LaoNLP/issues).

## Bug report

If you found the bug, you can create a issues at [https://github.com/wannaphong/LaoNLP/issues](https://github.com/wannaphong/LaoNLP/issues).
